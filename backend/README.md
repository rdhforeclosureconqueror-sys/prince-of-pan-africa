# Prince of Pan-Africa Backend

Core backend for the Prince of Pan-Africa platform.
Provides AI chat, voice, portal learning routes, and basic JWT auth routes.

## Endpoints
- `POST /auth/register`
- `POST /auth/login`
- `GET /auth/me`
- `GET /chat/ping`
- `POST /chat/message`
- `POST /chat/tts`
- `POST /chat/voice`
- `GET /portal/list`
- `POST /portal/start`
- `POST /portal/continue`
- `POST /portal/create_image`
- `POST /api/voice/tts`
- `POST /api/voice/stt`
- `GET /api/voice/health`
- `GET /health`
