# Mutual Aid Society System Design Draft

**Status:** Historical source draft converted to pointer  
**Activation status:** Building Toward Activation  
**Runtime status:** No runtime implementation is authorized by this file

This file previously held a single Mutual Aid Society system design draft. The Phase 1 documentation package now splits that material into six controlling documents so policy, operations, technical planning, language, and pilot readiness stay organized.

For the full Mutual Aid Society operating model, see [SIMBA_MUTUAL_AID_DOCS_INDEX.md](SIMBA_MUTUAL_AID_DOCS_INDEX.md).

No mutual aid distributions are authorized before activation. Support is reviewed and not guaranteed. The $20,000 threshold must be reached before activation, and activation also requires approved policy, governance process, accounting controls, privacy rules, and approval controls.
